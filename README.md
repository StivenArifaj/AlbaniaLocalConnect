# Albania Local Connect

> Your Professional Hub in Albania - Build Your Career & Find Local Services

Albania Local Connect is a comprehensive professional services platform that connects Albanian professionals, service providers, and clients. Whether you're building your career or looking for trusted local services, we've got you covered.

### Career Hub
- **Resume Builder** - Create ATS-friendly resumes with professional templates
- **Portfolio Creator** - Showcase your work with stunning portfolio pages
- **AI-Powered Content** - Get intelligent suggestions for resume content
- **Export to PDF** - Download professional documents ready for applications
- **Template Library** - Choose from modern, industry-specific templates

### Services Marketplace
- **Find Local Services** - Browse trusted service providers in your area
- **Book Appointments** - Schedule services with ease
- **Rate & Review** - Share your experiences and help the community
- **Manage Bookings** - Track all your appointments in one place
- **Provider Profiles** - Detailed information about service professionals

### Unified Platform
- **Single Account** - One login for all features
- **Professional Profile** - Showcase yourself as a job seeker or service provider
- **Integrated Dashboard** - Manage career documents and service bookings
- **Community Connection** - Network with other Albanian professionals

## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ and npm
- Modern web browser

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd albania-local-connect-main

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
# Create production build
npm run build

# Preview production build
npm run preview
```
